# SQL性能下降的原因

1. 查询语句问题；

2. 索引失效：

    - 单值索引：比如`create index idx_user_name on user(name)`
    - 复合索引：比如`create index idx_user_nameEmail on user(name, Email)`

3. 关联查询太多`join`(设计缺陷或者不得已的需求)；

4. 服务器调优以及各个参数的设置（缓存、线程数等）
